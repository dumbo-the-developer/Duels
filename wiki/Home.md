# Duels Wiki

Welcome to the **Duels** plugin documentation! This wiki provides comprehensive information about installing, configuring, and using the Duels plugin for your Minecraft server.

## Overview

Duels is an ultimate solution for server owners wanting to create a professional duel system. Players can challenge each other to 1v1 or party battles with customizable kits, arenas, and various betting options.

## Key Features

- **Custom Kits** - Create unlimited custom kits with items, armor, and effects
- **Arena System** - Set up multiple arenas with teleport positions
- **Queue System** - Automated matchmaking with ELO-based rating system
- **Party Duels** - Team-based duels with party system
- **Betting System** - Item betting and money betting (with Vault)
- **Spectator Mode** - Watch ongoing duels
- **Statistics Tracking** - Track wins, losses, ratings, and match history
- **PlaceholderAPI Support** - Extensive placeholder integration
- **Third-Party Hooks** - Integrations with popular plugins
- **GUI Customization** - Fully customizable inventory GUIs
- **Countdown System** - Configurable match countdowns
- **Rating System** - ELO-based skill rating for competitive play
- **Queue Signs** - Create clickable signs for quick queue joining

## Getting Started

1. **Install the Plugin** - Follow the [Installation Guide](Installation.md)
2. **Set up Lobby** - Use `/duels setlobby` to set the lobby spawn
3. **Create Kits** - Create your first kit with `/duels savekit <name>`
4. **Create Arenas** - Set up arenas with `/duels create <name>`
5. **Configure Settings** - Customize [config.yml](Configuration.md) to your needs
6. **Test the System** - Challenge a player with `/duel <player>`
