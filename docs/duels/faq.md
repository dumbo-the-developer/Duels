# FAQ

### Nothing happens when I run a command

* Check your permissions.
* Check console for startup errors.

### Players can’t be teleported into arenas

* Make sure arena spawns are set.
* Ensure the world is loaded.

### Duels won’t start / queue never pops

* Create at least one arena.
* Create at least one kit (if kits are required).
* Make sure requests/queue are enabled in config.
