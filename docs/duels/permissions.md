# Permissions

### Player permissions

* `duels.use` — basic usage
* `duels.queue` — queue usage

### Admin permissions

* `duels.admin` — admin commands

{% hint style="info" %}
If you use a permission plugin (LuckPerms), grant nodes to groups. Avoid giving `duels.admin` to default players.
{% endhint %}
